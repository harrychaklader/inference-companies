# Inference Companies

A curated tracker of **startup-focused AI inference companies** — covering headcount, funding stage, amount raised, and careers pages.

🌐 **Live page:** https://hsmahon.github.io/inference-companies/

---

## What's tracked

| Field | Description |
|---|---|
| **Company** | Startup name |
| **Headcount** | Employee count snapshot |
| **Funding Round** | Latest known series (Seed, Series A–E, Strategic, etc.) |
| **Amount Raised** | Dollar amount from the most recent disclosed round |
| **Careers Page** | Direct link to open roles |
| **Sources** | Public sources used for headcount and funding data |
